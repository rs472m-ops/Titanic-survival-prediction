# Titanic Survival Prediction using Hyperparameter Tuning

## Project Description

This project is a simple Machine Learning web application built using Streamlit.

The application predicts whether a passenger survived the Titanic disaster based on passenger information.

The project also demonstrates Hyperparameter Tuning using Random Forest Classifier.

## Features

- View Titanic Dataset
- Data Cleaning
- Label Encoding
- Train-Test Split
- Random Forest Model
- Hyperparameter Tuning
- Accuracy Evaluation
- Passenger Survival Prediction

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Run Project

Install libraries:

pip install -r requirements.txt

Run Streamlit App:

streamlit run app.py
